## *Description*
Presenting a well-crafted README.md file with your GitHub project is the first step in communicating that project to other developers and potential employers. ## *Installation*
`npm i inquirer` on the command line in root folder.

## *Usage*
Type `node index.js` in the terminal, then go through the prompts, pressing `ENTER` after each response.
